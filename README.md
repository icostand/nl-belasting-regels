# Dutch Tax Rules (Machine-Readable)

Open dataset of Dutch personal tax thresholds, rates, allowances, and calculator rules in JSON.
Maintained for use in calculators and financial tooling.

## Why
Dutch tax data is published across official pages and guidance notes in prose form.
This repo keeps it versioned, machine-readable, and diffable year over year.

## Coverage
- Income Tax / box rules
- Social contributions
- Payroll deductions
- Mortgage-related support rules
- Benefits and allowances where relevant to calculators

## Used by
These rules power the calculators at [Qalqlet](https://qalqlet.com/nl/),
where each result links back to its source and rule version.

## Sources
Every file carries a source field pointing to the primary official government page
and a last_reviewed date.
